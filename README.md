# Apex_Cart-Project
# 🛒 ApexCart – Full-Stack E-Commerce Product Catalog

A modern full-stack e-commerce product catalog built using React, Vite, Node.js, and Express. The application allows users to browse products, view product details, and add products to a shopping cart.

---

## 🚀 Features

- Product catalog with responsive UI
- Product detail page
- Shopping cart functionality
- React Router navigation
- REST API using Express.js
- Modern React Hooks
- Responsive design using Tailwind CSS
- Fast development with Vite

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- React Router DOM
- Tailwind CSS
- Lucide React Icons

### Backend
- Node.js
- Express.js

## ⚙️ Installation

### Clone the repository

```bash
git clone <repository-url>
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd backend
npm run dev
```

Backend runs on:

```
http://localhost:5000
```

### Start Frontend

```bash
cd frontend
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
